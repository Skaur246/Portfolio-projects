# Python Trip Mileage Calculator

## Overview
This project is a Python-based trip mileage calculator designed to practice programming logic and problem-solving. The program calculates trip distance and related values using user input, custom functions, and control structures without relying on advanced libraries.

The focus of this project is writing clear, logical, and well-structured Python code.

## Skills Demonstrated
- Python programming fundamentals
- Functions and modular design
- Conditional statements and loops
- Input validation
- Problem-solving with logic

## Features
- Accepts user input for trip details
- Calculates mileage values
- Uses reusable functions
- Handles invalid or missing input

## Tools
- Python
