# Linear Algebra Perceptron Classifier

## Overview
Built a binary perceptron classifier using linear algebra concepts such as z-score standardization, weight vectors, normalization, dot products, bias, and matrix transformations.

## Skills Demonstrated
- Linear Algebra for Machine Learning
- Perceptron fundamentals
- Data standardization (z-scores)
- Vector normalization
- Model evaluation

## Tools
- Excel
- Linear Algebra
